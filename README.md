📍 NearbyNow – Local Opportunity Finder (Frontend)

NearbyNow is a simple frontend web app that helps users discover nearby opportunities like part-time jobs, events, internships, and services — all in one place.
Built using HTML, CSS, and JavaScript, this project focuses on creating a clean, responsive, and user-friendly interface for local discovery.

🚀 Features

📍 View nearby opportunities (jobs, services)
🔍 Search and filter options
🖥️ Clean and modern UI
📱 Fully responsive design (mobile + desktop)
⚡ Fast and lightweight (no frameworks)

🛠️ Tech Stack

HTML – Structure
CSS – Styling & layout
JavaScript – Interactivity

Download or clone the repository
Open index.html in your browser
No installation required.

🎯 Purpose

This project is built as a frontend prototype to demonstrate how a local opportunity discovery platform can look and feel.
